# 🎫 Sistema DANFE - Gestão de Notas Fiscais e Boletos

Sistema completo para gestão de notas fiscais DANFE e boletos bancários, com funcionalidades de cadastro, edição, exclusão e exportação para Excel.

## ✨ Funcionalidades

- ✅ Cadastro completo de notas fiscais (NFe, fornecedor, produtos, valores)
- ✅ Gestão de boletos com múltiplas parcelas
- ✅ Status automático dos boletos (Pendente, Próximo Vencimento, Vencido, Pago)
- ✅ Edição e exclusão de notas e boletos
- ✅ Exportação para Excel (CSV)
- ✅ Interface responsiva e moderna
- ✅ PWA (Progressive Web App) - instalável em dispositivos móveis
- ✅ Funciona offline após primeira visita

## 🚀 Tecnologias Utilizadas

- HTML5
- CSS3 (Design moderno com gradientes e flexbox)
- JavaScript (Vanilla - sem frameworks)
- LocalStorage para persistência de dados
- PWA (Service Worker e Manifest)
